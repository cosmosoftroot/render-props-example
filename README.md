# Render props example

This is an example of how render props work. Enjoy with the cat!!!

## Features

- standard (linter)
- Husky
- Sass
- serve
- webpack

## scripts

build production version

    npm run build 

Development environment

    npm run dev

Code review

    npm run lint

Fix code    

    npm run lint:fix

Serve production application in port 4000

    npm run start
